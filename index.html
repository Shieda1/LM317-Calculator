What is the LM317 calculator?
The LM317 calculator is a tool that helps in choosing which external resistors to use to get the desired output voltage from the LM317 voltage regulator.
How to calculate output voltage?
To calculate output voltage you should use the following formula:
output voltage = 1.25 * (1 + (R1 / R2))
